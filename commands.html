Command Info Usage Example
ping Pong! -mod ping -mod ping
help Lists all the available commands and their usage. -mod help -mod help
warn Warns the user with the specified reason. -mod warn @user [reason] -mod warn @User Stop swearing.
mute Mutes the user with specified args. Both arguments can be omitted. -mod mute @user [reason] | [time] -mod mute @User profanity | 1 hour
unmute Unmutes the mentioned user. -mod unmute @user -mod unmute @User
kick Kicks the user with the specified reason. -mod kick @user [reason] -mod kick @User rude behaviour
ban Bans the user with specified args. Both arguments can be omitted. -mod ban @user [reason] | [time] -mod ban @User spamming | 3 months
unban Unbans the specified user. -mod unban @user -mod unban User#1234
